Under the Hood


SendDiscordMessage(webhookURL, message) {
    ; Sanitize the message to prevent JSON injection
    StringReplace, message, message, \, \\, All
    StringReplace, message, message, ", \", All
    StringReplace, message, message, `r`n, \n, All
    StringReplace, message, message, `n, \n, All

    payload := "{""content"": """ . message . """}"
    
    ComObjError(false)
    http := ComObjCreate("WinHttp.WinHttpRequest.5.1")
    
    try {
        http.Open("POST", webhookURL, false)
        http.SetRequestHeader("Content-Type", "application/json")
        http.Send(payload)
        
        responseStatus := http.Status
        if (responseStatus = 204 or responseStatus = 200) {
            return true
        } else {
            return false
        }
    } catch e {
        return false
    }
}

This is a snippet of the core function that sends messages to Discord. It handles message sanitization, payload creation, and HTTP requests.
